Simple Calculator in console without IF, SWITCH, TERNARY OPERATOR.

Only for integers for arithmetic operations + , -, m, /.
Attention!
Instead of the character < * > use < m >.

Start program:
1. Start PowerShell;
2. Change the directory where you are located file Gate.class;
3. Enter: java Gate;
4. Enter first argument (integer) + space key;
5. Enter second argument (sign) + space key;
6. Enter third argument (integer) + enter key.

Attention!
Instead of the character < * > use < m >.
 
Example :

java Gate 2 + 3
Result=5
Result with different method=5

java Gate 2 m 3
Result=6
Result with different method=6
